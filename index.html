Software tool development

GALE
General Adsorbent Library and Evaluation (GALE) is a MATLAB-based software providing GUI for the adsorbent parameter library and for the evaluation of adsorbents.

Pyapep
Python-based Adsorpton Process Evaluation Pacakage(Pyapep) is a comprehensive python package covering from fitting the isotherm data to simulating adsorption processes for separation/storage.

NH3PE
This web application (Web App) provides parameter estimation studies with the data from ammonia synthesis experiments. With simple click and typing, you can obtain the reaction parameters of your experiment data.

His Software/Web Apps

How we can
work together

Make your research distributable !
Seongbin can provide a comprehensive analysis of chemical processes based on the sophisticated simulations. For Seongbin, such analysis can be a crucial idea for a practical software tool applicable in a variety of studies. The software tool can be in the form of the following examples:

GUI

GALE: a Simulator with GUI

This software provides graphical user interface (GUI) for the adsorbent evaluation. The users can intuitively understand the evaluation procedure and implement the evaluation to the GUI.

Package for CLI

Pyapep: python package

Pyapep is a python package for the simulation of various adsorpton processes such as pressure swing adsorption (PSA) process or adsorption storage tank. Combined with diverse tools in python, the users can build a variety of computational approaches. In other command-line interface (CLI) language, we can also develop another package, later.

Web App

NH3PE: web App

With neither any setting nor additional download, this provides the parameter estimation work for the ammonia synthesis reaction. By uploading simple csv files to the website, the users can obtain the reaction kinetic parameters for different catalysts.
